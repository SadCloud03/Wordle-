## Practice of coding skills by making of the game WORDLE

### About WORDLE:
Wordle is a web_based word game created and developed by the Welsh software engineer Josh Wardle. In the game, players have six attempts to guess a five-letter word, receiving feetback through colored tiles that indicate correct letters and their placement. 

## this is just a little practice of coding skills in python.

## Features
* N-letter word guessing logic.
* Colored-coded feedback (Correct, Misplaced, Wrong).
* N attempts per game.

## Presequisites
* Python 3.x
